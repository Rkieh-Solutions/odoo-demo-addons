# WoodCut Pro

A professional woodworking cutting optimization tool.

## Features
- **Guillotine Packing Algorithm**: Generates edge-to-edge cuts compatible with workshop saws.
- **Multi-Board Support**: Automatically splits pieces across multiple boards if needed.
- **Kerf Awareness**: Accounts for saw blade thickness in all calculations.
- **Piece Rotation**: Optional rotation to maximize board yield.
- **Step-by-Step Instructions**: Clear list of cuts with (X, Y) positions.
- **Printable Reports**: Generate workshop-ready PDF/Print layouts.

## How to Run
Simply open `index.html` in any modern web browser. No installation or build steps required.

## Tech Stack
- React 18
- Tailwind CSS
- Lucide Icons
- Babel (for in-browser JSX transformation)
